What is the buffer capacity?
Buffer capacity is the number of moles of a base or acid that should be added to one liter of a buffer in order to cause it's pH to change by 1 unit.
How to calculate buffer capacity?
To calculate buffer capacity you should use the following formula:
pH = pKa + log10([A^-]/[HA])
where:
[A^-] = the concentration of a base in the buffer
[HA] = the concentration of an acid in the buffer
Why use buffer capacity?
The buffer capacity is used to quantify the ability of a solution to resist the changes in pH by either absorbing desorbing or absorbing H+ and OH- ions.
